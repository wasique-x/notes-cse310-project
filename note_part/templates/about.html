About


Notes is a minimal app written in Django that is intended for taking takes. Notes implements CRUD functionality of Notes along with multi User interactions.


Features

    Users can View and Select their own Notes.

    Users can Create new notes having Title and Body.

    Users can Modify, Trash, Archive and Delete existing Notes.

    Additionally, Trashed Notes can be Untrashed and Archived Notes can be Unarchive.

    Users can Register, Login and Logout.

    Multiple Users can simultaneously access the App
